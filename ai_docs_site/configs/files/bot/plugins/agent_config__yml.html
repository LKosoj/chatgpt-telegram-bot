bot/plugins/agent_config.yml

Источник: bot/plugins/agent_config.yml

Файл конфигурации задаёт параметры подключения к языковой модели, описывает поведение и инструментарий двух автономных агентов, и устанавливает параметры хранения данных и журналирования.

Секции и ключи

openai — Параметры подключения к LLM-шлюзу через OpenAI-совместимый API.
openai/model — Идентификатор модели, маршрутизируемый через шлюз.
openai/base_url — URL-адрес эндпоинта API с указанием версии.
openai/temperature — Параметр температуры генерации, нулевое значение обеспечивает детерминированный вывод.
openai/max_tokens — Максимальное количество токенов в ответе.
openai/request_timeout — Таймаут ожидания ответа от API в секундах.
agents — Описание ролей и конфигураций автономных агентов.
agents/researcher — Агент для сбора и анализа информации с выдачей результатов в формате JSON на русском языке.
agents/researcher/system_prompt — Системная инструкция, определяющая роль и формат вывода агента.
agents/researcher/max_iterations — Предельное число итераций работы агента.
agents/researcher/max_execution_time — Лимит времени выполнения в секундах.
agents/researcher/tools — Перечень инструментов агента-исследователя.
agents/researcher/tools/ddg_web_search — Инструмент поиска в интернете через DuckDuckGo с заданным регионом и лимитом результатов.
agents/researcher/tools/conversation_analytics — Инструмент анализа тональности и содержания текста.
agents/planner — Агент для создания структурированных планов с конкретными измеримыми шагами на русском языке.
agents/planner/system_prompt — Системная инструкция, определяющая роль планировщика и требования к планам.
agents/planner/max_iterations — Предельное число итераций работы агента.
agents/planner/max_execution_time — Лимит времени выполнения в секундах.
agents/planner/tools — Перечень инструментов агента-планировщика.
agents/planner/tools/agent_tools/manage_plan_tasks — Инструмент для управления задачами плана.
agents/planner/tools/agent_tools/ask_telegram_user — Инструмент для уточнения требований у пользователя через Telegram.
database — Параметры локального хранилища данных.
database/path — Путь к файлу базы данных SQLite.
database/connection_timeout — Таймаут установления соединения в секундах.
logging — Настройки журналирования.
logging/level — Уровень детализации логов.
logging/format — Шаблон форматирования записей лога с указанием времени, имени логгера, уровня и сообщения.

Важные параметры

openai/max_tokens — Значение 200000 обеспечивает обработку объёмных контекстов и длинных ответов.
openai/temperature — Нулевое значение исключает случайность в генерации, критично для структурированного вывода JSON агентом-исследователем.
agents/researcher/max_execution_time — Лимит 300 секунд компенсирует потенциальные задержки при веб-поиске и анализе.
agents/planner/tools/agent_tools/ask_telegram_user — Интеграция с Telegram обеспечивает обратную связь с пользователем в процессе планирования.
