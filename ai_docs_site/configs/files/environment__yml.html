environment.yml

Источник: environment.yml

Файл описывает окружение Conda с именем myenv, которое формирует среду выполнения для асинхронного Telegram-бота с расширенными возможностями: голосовое взаимодействие, работа с нейросетевыми API, веб-скрапинг, обработка изображений и аудио, а также интеграция с внешними сервисами поиска и вычислений. Структура включает три канала пакетов и двухуровневую систему зависимостей — базовые через conda и дополнительные через pip.

Секции и ключи

name — идентификатор создаваемого окружения Conda, значение myenv.
channels — упорядоченный список источников пакетов: приоритетный канал defaults, сообщественный conda-forge и специализированный pytorch для фреймворков глубокого обучения.
dependencies — перечень пакетов для установки, включающий как прямые conda-зависимости, так и вложенную секцию pip с пакетами, недоступными в каналах Conda или требующими специфических версий.
pip — вложенная подсекция внутри dependencies, содержащая пакеты для установки через менеджер pip с возможностью точного управления версиями и доступом к пакетам из PyPI.

Важные параметры

pydub~=0.25.1 — библиотека обработки аудио с фиксацией минорной версии через совместимостный оператор.
pytube~=15.0.0 — инструмент загрузки видео с YouTube с ограничением обновлений в рамках мажорной версии.
gtts~=2.5.1 — сервис текст-в-речь от Google с контролем совместимости.
youtube-transcript-api=0.6.2 — жёсткая привязка к конкретной версии API для извлечения субтитров.
readability-lxml=0.8.1 — экстрактор читаемого контента из HTML с фиксированной версией.
lxml-html-clean=0.1.1 — отдельный пакет очистки HTML, выделенный из основной библиотеки lxml.
accelerate — библиотека оптимизации распределённого обучения моделей Hugging Face, присутствующая несмотря на закомментированные зависимости transformers и pytorch.
openai-whisper — система распознавания речи от OpenAI, установливаемая через pip вместо conda.
clip — модальная модель сопоставления изображений и текста от OpenAI.
annoy — библиотека приближённого поиска ближайших соседей для векторных индексов.
