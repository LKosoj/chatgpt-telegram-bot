Архитектура runtime

Точка входа и инициализация

bot/__main__.py
    ↓
PluginManager → Database → OpenAIHelper → ChatGPTTelegramBot

Обязательные переменные окружения: - TELEGRAM_BOT_TOKEN - OPENAI_API_KEY

Telegram-polling

  • Модуль: bot/telegram_bot.py
  • Базовый URL: http://localhost:8081/bot

Поток обработки запросов

Компонент Ответственность
telegram_bot.py Обработка обновлений Telegram
openai_helper.py API-вызовы к LLM
openai_tool_handler.py Выполнение tool calls
plugin_manager.py Плагины и валидация
database.py SQLite, сессии, история
chat_modes_registry.py + chat_modes.yml Режимы чата

Плагины и инструменты

Базовый класс

  • Файл: bot/plugins/plugin.py
  • Идентификация: plugin_id
  • Неймспейсинг функций: function_prefix

Загрузка и валидация

Параметр Назначение
PLUGINS CSV allow-list разрешённых плагинов
PLUGIN_STRICT_VALIDATION Строгая/мягкая валидация дубликатов

Правила: - Загрузка из bot/plugins/*.py - Пакетное выполнение: asyncio.gather - Инъекция chat_id/user_id в аргументы функции - Прямой ответ плагина обходит повторный вызов модели - Различие форматов спецификаций для Google и прочих моделей

Режимы чата

Декларация: bot/chat_modes.yml

Поле Назначение
tools Ограничение доступных плагинов (по умолчанию ['All'])
— Имена модулей плагинов, не plugin_id

Валидация: tool-ссылки проверяются при инициализации.

Обработчики Telegram

  • Нормализация команд: PluginManager.get_plugin_commands()
  • Запрет: пробелы в именах команд
  • Обработка фильтров сообщений без eval

База данных

Параметр Значение Назначение

SQLITE_BUSY_TIMEOUT_MS — Таймаут при занятости
busy_timeout — Аналог на уровне соединения

Docker-развёртывание

docker-compose.yml

services:
  chatgpt-telegram-bot:
    build:
      context: .
      dockerfile: Dockerfile
    volumes:
      - .:/app        # Горячая перезагрузка при разработке
    restart: unless-stopped

Политика unless-stopped: автоматический перезапуск при сбоях или перезагрузке системы, кроме ручной остановки.

Chat Modes Registry

Валидация (STRICT_MODE=true)

  • Уникальность id
  • Разрешённые model_profile
  • Корректность ссылок на плагины
  • Валидность regex

При STRICT_MODE=true: ошибки блокируют старт бота.

Структура ChatMode

Поле Содержимое
system_prompt Системный промпт
model_profile Маршруты LLMGateway: high/light/big_context/vision/image_generation/image_edit/transcription/tts
plugins_allowlist/plugins_denylist Фильтры плагинов
budget Ограничения
feature-флаги vision, audio, web_search, deep_research, agent_runtime, hindsight, skills, mcp
agent_limits Лимиты агента
tts_voice Голос TTS
response_format Формат ответа

Изменения требуют перезапуска.
