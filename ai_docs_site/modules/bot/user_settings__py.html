bot/user_settings

Источник: bot/user_settings.py

Модуль содержит константы ключей пользовательских настроек и вспомогательные функции для безопасного чтения, нормализации и изменения настроек. Основное назначение — абстрагировать работу с хранилищем пользовательских предпочтений и обеспечить консистентность форматов данных, особенно для списков строковых идентификаторов.

Ключевые структуры данных USER_LANGUAGE_SETTING — Ключ настройки языка пользователя (str: "language"). USER_TTS_MODEL_SETTING — Ключ настройки модели TTS (str: "tts_model"). USER_TTS_VOICE_SETTING — Ключ настройки голоса TTS (str: "tts_voice"). USER_DISABLED_PLUGINS_SETTING — Ключ списка отключённых плагинов (str: "disabled_plugins"). USER_DISABLED_SKILLS_SETTING — Ключ списка отключённых навыков (str: "disabled_skills").

ensure_settings_dict(settings: Any) -> dict[str, Any] Гарантирует, что значение является словарём. Аргументы settings — Входное значение произвольного типа. Возвращает Исходный словарь, если settings имеет тип dict, иначе пустой словарь.

normalize_string_list(value: Any) -> list[str] Нормализует произвольное значение в отсортированный список уникальных непустых строк. Аргументы value — Входное значение, ожидается список. Возвращает Отсортированный список уникальных строк без дубликатов и пустых элементов. Для несписковых входных данных возвращает пустой список.

get_user_settings(db: Any, user_id: int | None) -> dict[str, Any] Извлекает настройки пользователя из объекта базы данных. Аргументы db — Объект базы данных с методом get_user_settings. user_id — Идентификатор пользователя или None для анонимного пользователя. Возвращает Словарь пользовательских настроек или пустой словарь при отсутствии user_id, метода get_user_settings или пустого результата.

set_disabled_value(settings: dict[str, Any], key: str, value: str, disabled: bool) -> None Модифицирует множество отключённых элементов в настройках. Аргументы settings — Словарь настроек для изменения in-place. key — Ключ списка отключённых элементов. value — Идентификатор элемента для добавления или удаления. disabled — Флаг: True для добавления в список отключённых, False для удаления.
