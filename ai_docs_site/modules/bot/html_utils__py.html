bot/html_utils

Источник: bot/html_utils.py

Комплексный модуль для создания расширенной HTML-визуализации с поддержкой Markdown, Mermaid-диаграмм, PlantUML, библиографических ссылок и интерактивного клиентского интерфейса. Модуль реализует полный конвейер обработки: извлечение и защита Mermaid-диаграмм через плейсхолдеры, Markdown-конвертацию с помощью markdown2, многоуровневую постобработку HTML для восстановления диаграмм и исправления URL, генерацию интерактивных HTML-контейнеров с панелями управления, а также CSS-стили и JavaScript-функционал для масштабирования, экспорта и переключения тем.

Ключевые структуры данных - HTMLVisualizer — основной класс для конвертации текста в HTML с обработкой диаграмм и ссылок - mermaid_blocks — словарь {block_id: original_content} для сохранения Mermaid-кода до Markdown-преобразования - mermaid_containers — словарь {container_id: container_html} для восстановления HTML-обёрток диаграмм - mermaid_placeholders — словарь {temp_id: original_placeholder} для промежуточной замены плейсхолдеров - reference_links — словарь {ref_num: {'url': str, 'text': str}} сопоставления библиографических сносок - html_content — список строк HTML-разметки, накапливаемый для финального отчёта - plot_files — список имён файлов графиков PNG - mermaid_files — отфильтрованный и отсортированный список файлов .mermaid для текущей сессии
