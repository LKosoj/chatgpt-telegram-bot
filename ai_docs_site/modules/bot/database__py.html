bot/database

Источник: bot/database.py

Модуль предоставляет потокобезопасный singleton-класс Database для работы с SQLite. Реализует хранение пользовательских настроек, контекста разговоров с поддержкой сессий, информации об изображениях и фоновых задач finalize. Каждый поток получает собственное соединение через thread-local storage; транзакции управляются через контекстный менеджер с автоматическим commit/rollback.

Ключевые структуры данных Database._local — threading.local хранилище для thread-specific соединений с SQLite

class Database Потокобезопасный singleton для работы с SQLite базой данных пользовательских данных Поля _instance — единственный экземпляр класса (singleton) _lock — threading.Lock для защиты создания экземпляра _connection_lock — threading.Lock для защиты создания новых соединений _local — threading.local для хранения соединений по потокам db_path — путь к файлу базы данных _op_lock — threading.RLock для защиты операций в рамках одного соединения Методы new(cls) — Создаёт или возвращает singleton-экземпляр с инициализацией пути БД и вызовом init_db init(self) — Пустой метод-заглушка для предотвращения повторной инициализации при множественных вызовах get_connection(self) -> Generator[sqlite3.Connection, None, None] — Контекстный менеджер для потокобезопасного доступа к соединению del(self) — Закрывает thread-local соединение при удалении объекта init_db(self) — Инициализирует схему базы данных с таблицами user_settings, conversation_context, images, hindsight_finalize_jobs и индексами save_user_settings(self, user_id: int, settings: Dict[str, Any]) -> None — Сохраняет пользовательские настройки в формате JSON с upsert-логикой get_user_settings(self, user_id: int) -> Optional[Dict[str, Any]] — Возвращает пользовательские настройки по ID или None при отсутствии get_active_session_id(self, user_id: int) -> Optional[str] — Возвращает ID активной сессии пользователя или None _session_name_source_text(content: Any) -> str — Статический метод для извлечения текстового содержимого из строки или списка словарей

new(cls) Создаёт или возвращает singleton-экземпляр с инициализацией пути БД и вызовом init_db

init(self) Пустой метод-заглушка для предотвращения повторной инициализации при множественных вызовах

get_connection(self) -> Generator[sqlite3.Connection, None, None] Контекстный менеджер для потокобезопасного доступа к соединению с базой данных Аргументы self — экземпляр Database Возвращает sqlite3.Connection — соединение с настроенными PRAGMA (row_factory=sqlite3.Row, foreign_keys=ON, journal_mode=WAL, busy_timeout) Исключения Exception — при ошибке выполняется rollback и исключение пробрасывается далее

del(self) Закрывает thread-local соединение при удалении объекта

init_db(self) Инициализирует схему базы данных с таблицами user_settings, conversation_context, images, hindsight_finalize_jobs и индексами

save_user_settings(self, user_id: int, settings: Dict[str, Any]) -> None Сохранение пользовательских настроек в таблицу user_settings Аргументы user_id — идентификатор пользователя settings — словарь настроек для сериализации в JSON Возвращает None Исключения Exception — ошибка сериализации или выполнения SQL; логируется и пробрасывается

get_user_settings(self, user_id: int) -> Optional[Dict[str, Any]] Получение пользовательских настроек из таблицы user_settings Аргументы user_id — идентификатор пользователя Возвращает Dict[str, Any] — десериализованные настройки при наличии записи; None — если запись не найдена Исключения Exception — ошибка десериализации или выполнения SQL; логируется и пробрасывается

get_active_session_id(self, user_id: int) -> Optional[str] Получение идентификатора активной сессии пользователя Аргументы user_id — идентификатор пользователя Возвращает str — session_id активной сессии (is_active = 1); None — если активная сессия отсутствует

_session_name_source_text(content: Any) -> str Статический метод для извлечения текстового содержимого из различных типов данных Аргументы content — строка или список словарей с полем 'text' Возвращает str — извлечённый текст или пустая строка
