bot/openai_helper

Источник: bot/openai_helper.py

Модуль реализует центральный компонент для работы с языковыми моделями — инициализацию клиентов, управление историей диалогов, интеграцию с системой плагинов, долгосрочную память (Hindsight), классификацию намерений пользователя, обработку изображений, TTS, транскрипцию и вспомогательные функции для обработки ответов моделей и управления лимитами токенов.

Ключевые структуры данных

OpenAIHelper — основной класс-обёртка для взаимодействия с LLM API, управления диалогами и плагинами. self.conversations — словарь {chat_id: [messages...]} хранения истории сообщений self.conversations_vision — словарь {chat_id: bool} флагов использования vision-режима self.loaded_conversation_sessions — словарь {chat_id: session_id} отслеживания загруженных сессий self.last_updated — словарь {chat_id: datetime} временных меток последней активности self._tts_models_cache — кэш кортеж (timestamp, [model_names]) доступных TTS-моделей self._tts_voices_cache — кэш {model: (timestamp, [voice_names])} голосов по моделям TTS self.chat_modes_registry — реестр режимов чата из YAML-конфигурации common_args — словарь параметров для вызова chat.completions.create, формируемый динамически в зависимости от модели tools — список спецификаций функций для function calling, получаемый от plugin_manager

default_max_tokens(model: str = None) -> int Возвращает стандартное максимальное количество токенов для указанной модели. Аргументы model — имя модели; если не указано, используется значение по умолчанию Возвращает количество токенов (int)

are_functions_available(model: str) -> bool Проверяет, поддерживает ли указанная модель вызов функций (function calling). Аргументы model — имя модели для проверки Возвращает True, если модель поддерживает функции, иначе False

class OpenAIHelper Основной класс для взаимодействия с OpenAI-совместимым API и LLM Gateway. Поля client — асинхронный клиент openai.AsyncOpenAI gateway_client — клиент для работы с LLM Gateway config — словарь с конфигурацией plugin_manager — менеджер плагинов db — экземпляр базы данных conversations — словарь историй диалогов {chat_id: [messages...]} hindsight_client — клиент HindsightClient или None Методы init(self, config: dict, plugin_manager: PluginManager, db: Database) — инициализация клиентов, валидация конфигурации и настройка подключений
