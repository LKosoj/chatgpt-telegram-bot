bot/plugins/show_me_diagrams

Источник: bot/plugins/show_me_diagrams.py

Плагин для генерации различных диаграмм и визуализаций с использованием PlantUML.

Ключевые структуры данных diagram_types — словарь маппинга строковых идентификаторов типов диаграмм на соответствующие методы-генераторы PlantUML-кода diagram_prompts — словарь системных промптов с шаблонами PlantUML-кода для каждого типа диаграммы type_hints — словарь подсказок для уточнения описания диаграммы пользователем

class ShowMeDiagramsPlugin(Plugin) Плагин для генерации различных диаграмм и визуализаций с использованием PlantUML. Поля diagram_types — словарь маппинга типов диаграмм на методы-генераторы plantuml_jar — путь к JAR-файлу PlantUML diagram_prompts — словарь системных промптов для генерации PlantUML-кода type_hints — словарь подсказок для уточнения описания диаграммы пользователем Методы init(self) — инициализация плагина, настройка типов диаграмм, пути к PlantUML, промптов и подсказок get_source_name(self) -> str — возвращает имя источника плагина get_spec(self) -> List[Dict] — возвращает спецификацию функций плагина для регистрации execute(self, function_name, helper, **kwargs) -> Dict — основной метод выполнения генерации диаграммы

get_source_name(self) -> str Возвращает имя источника плагина. Возвращает Строка "Show Me Diagrams"

get_spec(self) -> List[Dict] Возвращает спецификацию функций плагина для регистрации в системе. Возвращает Список из одного словаря с описанием функции generate_diagram, включающей параметры type, description и title

execute(self, function_name, helper, kwargs) -> Dict Основной метод выполнения генерации диаграммы. Обрабатывает выбор типа диаграммы, сбор описания от пользователя, генерацию PlantUML-кода, рендеринг изображения и возврат результата. Аргументы function_name — имя вызываемой функции helper — вспомогательный объект для взаимодействия с чатом kwargs — параметры вызова: type (тип диаграммы), description (описание), title (заголовок), user_id (идентификатор пользователя) Возвращает Словарь с результатом операции: либо {"result": "...", "image": "..."} при успехе, либо {"result": сообщение об ошибке} при неудаче Исключения Может вызывать исключения при ошибках выполнения PlantUML или работы с файловой системой

async def generate_diagram(self, diagram_type: str, title: str, description: str, helper, user_id: int) -> dict Генерирует диаграмму указанного типа и возвращает результат в формате direct_result. Аргументы diagram_type — строковый идентификатор типа диаграммы из diagram_types title — заголовок диаграммы description — описание содержания диаграммы (может быть пустым, тогда запрашивается у LLM) helper — экземпляр помощника с методами get_chat_response и ask user_id — идентификатор пользователя для контекста LLM Возвращает словарь с ключом "direct_result" содержащим kind, format, value (путь к файлу), add_value (PlantUML код) или "result" с ошибкой Исключения возвращает словарь с ошибкой при неудаче генерации, не выбрасывает исключения наружу

async def _generate_plantuml(self, puml_content: str, helper, user_id: int) -> tuple[str, str] Компилирует PlantUML код в PNG изображение с автоматическим исправлением ошибок. Аргументы puml_content — строка с кодом PlantUML helper — экземпляр помощника с методом ask user_id — идентификатор пользователя для LLM-исправлений Возвращает кортеж (puml_content, output_file) — исходный или исправленный код и путь к сгенерированному PNG Исключения Exception — если не удалось сгенерировать диаграмму после 3 попыток или файл не создан

async def _generate_diagram_code(self, diagram_type: str, title: str, description: str, helper, user_id: int) -> str Формирует PlantUML код через LLM по шаблону промпта. Аргументы diagram_type — строковый идентификатор типа диаграммы title — заголовок диаграммы description — описание содержания helper — экземпляр помощника с методом ask user_id — идентификатор пользователя Возвращает строку с сгенерированным PlantUML кодом

_generate_gantt_chart(self, description: str, title: str) -> str Генерирует PlantUML-код для диаграммы Ганта. Аргументы description — подробное описание содержимого диаграммы title — заголовок диаграммы Возвращает Строка с PlantUML-кодом диаграммы Ганта

async def _generate_gantt_chart(self, title: str, description: str, helper, user_id: int) -> tuple[str, str] Генерирует диаграмму Ганта. Аргументы title — заголовок диаграммы description — описание проекта и задач helper — экземпляр помощника user_id — идентификатор пользователя Возвращает результат _generate_plantuml — кортеж (код, путь к файлу)

_generate_mind_map(self, description: str, title: str) -> str Генерирует PlantUML-код для интеллект-карты (mind map). Аргументы description — подробное описание содержимого карты title — заголовок карты Возвращает Строка с PlantUML-кодом интеллект-карты

async def _generate_mind_map(self, title: str, description: str, helper, user_id: int) -> str Создаёт диаграмму mind map (интеллект-карту). Аргументы title — заголовок интеллект-карты description — описание центральной темы и ветвей helper — вспомогательный объект генерации user_id — идентификатор пользователя Возвращает Результат рендеринга интеллект-карты.

_generate_flowchart(self, description: str, title: str) -> str Генерирует PlantUML-код для блок-схемы. Аргументы description — подробное описание процесса для блок-схемы title — заголовок блок-схемы Возвращает Строка с PlantUML-кодом блок-схемы

async def _generate_flowchart(self, title: str, description: str, helper, user_id: int) -> tuple[str, str] Генерирует блок-схему (flowchart). Аргументы title — заголовок диаграммы description — описание процесса helper — экземпляр помощника user_id — идентификатор пользователя Возвращает результат _generate_plantuml — кортеж (код, путь к файлу)

_generate_project_timeline(self, description: str, title: str) -> str Генерирует PlantUML-код для временной шкалы проекта. Аргументы description — подробное описание событий и этапов title — заголовок временной шкалы Возвращает Строка с PlantUML-кодом временной шкалы

async def _generate_project_timeline(self, title: str, description: str, helper, user_id: int) -> str Создаёт временную шкалу проекта. Аргументы title — название проекта/шкалы description — описание этапов, задач и временных рамок helper — вспомогательный объект генерации user_id — идентификатор пользователя Возвращает Результат рендеринга временной шкалы.

_generate_infographic(self, description: str, title: str) -> str Генерирует PlantUML-код для инфографики. Аргументы description — подробное описание содержимого инфографики title — заголовок инфографики Возвращает Строка с PlantUML-кодом инфографики

async def _generate_infographic(self, title: str, description: str, helper, user_id: int) -> str Создаёт инфографику. Аргументы title — заголовок инфографики description — описание визуализируемых данных и структуры helper — вспомогательный объект генерации user_id — идентификатор пользователя Возвращает Результат рендеринга инфографики.

async def _generate_org_chart(self, title: str, description: str, helper, user_id: int) -> str Создаёт организационную структуру. Аргументы puml_code — строка с корректным PlantUML-кодом helper — вспомогательный объект для рендеринга (тип определяется контекстом) user_id — идентификатор пользователя, запросившего генерацию Возвращает Строка с результатом рендеринга (URL изображения, путь к файлу или base64-данные в зависимости от реализации helper).
