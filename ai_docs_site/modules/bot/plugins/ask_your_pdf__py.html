bot/plugins/ask_your_pdf

Источник: bot/plugins/ask_your_pdf.py

Модуль ask_your_pdf реализует плагин AskYourPDFPlugin и инструмент AskYourPDFTool для извлечения текста из PDF-документов и его анализа через OpenAI API / LLM.

Ключевые структуры данных cache_metadata.json — JSON-файл с полями files (словарь хеш→{size, last_accessed}) и total_size для управления LRU-очисткой кэша. cached_data — словарь {"cached_at": float, "result": Dict} для записи в файл кэша.

class AskYourPDFPlugin Плагин для извлечения и анализа содержимого PDF-файлов с продвинутым кэшированием. Поля temp_dir — директория для временных PDF-файлов. cache_dir — директория для файлов кэша извлечённого текста и результатов анализа. cache_metadata_path — путь к JSON-файлу метаданных кэша. max_cache_size_mb — максимальный размер кэша в мегабайтах (по умолчанию 500). max_cache_age_days — максимальный возраст записи кэша в днях (по умолчанию 10). max_extracted_text_chars — ограничение длины извлечённого текста (по умолчанию 50000). Методы __init__() — инициализирует директории и метаданные кэша. initialize(openai=None, bot=None, storage_root: str | None = None) -> None — перенастраивает пути при наличии корневого хранилища. get_source_name() -> str — возвращает идентификатор источника "AskYourPDF". get_spec() -> List[Dict] — возвращает JSON Schema спецификации функций analyze_pdf и upload_pdf. _init_cache_metadata() — создаёт начальный файл метаданных кэша при отсутствии. _update_cache_metadata(file_hash, file_size) — обновляет метаданные, удаляет устаревшие и превышающие лимит записи. generate_file_hash(file_path: str) -> str — вычисляет MD5-хеш по первым/последним 1024 байтам, размеру и mtime файла. _cache_key(file_hash: str, query: str) -> str — формирует ключ кэша из хеша файла и SHA256 запроса. _cache_file_path(file_hash: str, query: str) -> str — возвращает полный путь к файлу кэша. load_cache(file_hash: str, query: str) -> Dict — загружает результат из кэша при наличии и валидности по возрасту. save_cache(file_hash: str, query: str, result: Dict) — сохраняет результат в кэш с обновлением метаданных. _is_path_in_controlled_storage(file_path: str) -> bool — проверяет, находится ли путь в разрешённых директориях. _extract_text_with_pypdf2(file_path: str) -> str — извлекает текст через PyPDF2.PdfReader.

class AskYourPDFTool Инструмент для загрузки и анализа PDF-документов через LLM. Поля max_extracted_text_chars — максимальное количество символов извлечённого текста, передаваемое в промпт анализа. cache_dir — директория для хранения кешированных результатов анализа. Методы __init__(self, config: Dict, **kwargs) — инициализирует инструмент с конфигурацией и настройками кеширования. _is_path_in_controlled_storage(self, file_path: str) -> bool — проверяет, находится ли путь в пределах контролируемого хранилища. generate_file_hash(self, file_path: str) -> str — вычисляет SHA-256 хеш файла для использования в качестве ключа кеша. load_cache(self, file_hash: str, query: str) -> Optional[Dict] — загружает кешированный результат анализа по хешу файла и запросу. save_cache(self, file_hash: str, query: str, result: Dict) -> None — сохраняет результат анализа в кеш. _extract_text_with_pypdf2(self, file_path: str) -> str — извлекает текст из PDF с использованием библиотеки PyPDF2. _extract_text_with_textract(self, file_path: str) -> str — извлекает текст из PDF с использованием библиотеки textract. _extract_text_with_pdfminer(self, file_path: str) -> str — извлекает текст из PDF с использованием библиотеки pdfminer.six. _extract_pdf_text(self, file_path: str) -> str — последовательно пытается извлечь текст из PDF через доступные экстракторы; выбрасывает ValueError при неудаче. _build_analysis_prompt(self, file_path: str, query: str, extracted_text: str) -> str — формирует промпт для LLM-анализа на основе метаданных файла и извлечённого текста. execute(self, function_name: str, helper, **kwargs) -> Dict — выполняет вызванную функцию инструмента (upload_pdf или analyze_pdf); перехватывает все исключения и возвращает их как ошибку в ответе.
