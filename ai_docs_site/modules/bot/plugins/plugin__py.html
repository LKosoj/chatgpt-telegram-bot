bot/plugins/plugin

Источник: bot/plugins/plugin.py

Абстрактный базовый класс Plugin определяет интерфейс для создания плагинов к ChatGPT API. Предоставляет механизмы идентификации плагина, локализации, жизненного цикла и интеграции с Telegram-ботом через команды, обработчики сообщений и inline-запросов.

Ключевые структуры данных Plugin — абстрактный базовый класс, реализующий интерфейс плагина с поддержкой инициализации, локализации и регистрации обработчиков

class Plugin(ABC) Абстрактный базовый класс для создания плагинов ChatGPT API. Поля plugin_id — стабильный идентификатор плагина (по умолчанию None, используется имя класса) function_prefix — префикс пространства имён функций (по умолчанию None, используется plugin_id) openai — объект OpenAI API, устанавливается при инициализации bot — объект бота, устанавливается при инициализации storage_root — путь к корневой директории хранилища, устанавливается при инициализации Методы get_plugin_id() -> str — возвращает стабильный идентификатор плагина get_function_prefix() -> str — возвращает префикс пространства имён функций initialize(openai=None, bot=None, storage_root: str | None = None) -> None — инициализирует плагин с внешними зависимостями close() -> None — освобождает ресурсы при завершении работы плагина on_startup(application: Any) -> None — асинхронный хук, вызываемый после готовности Telegram-приложения get_bot_language() -> str — определяет язык бота из конфигурации OpenAI t(key: str, kwargs: Any) -> str — возвращает локализованный текст по ключу с подстановкой параметров get_source_name() -> str — возвращает название источника плагина (абстрактный) get_spec() -> [Dict] — возвращает спецификации функций в формате JSON Schema (абстрактный) execute(function_name: str, helper: Any, kwargs: Optional[Dict[str, Any]]) -> Dict — выполняет функцию плагина и возвращает JSON-сериализуемый результат (абстрактный) get_commands() -> List[Dict] — возвращает список поддерживаемых Telegram-команд get_message_handlers() -> List[Dict] — возвращает список обработчиков сообщений get_inline_handlers() -> List[Dict] — возвращает список обработчиков inline-запросов

get_plugin_id() -> str Возвращает стабильный идентификатор плагина. Возвращает Значение plugin_id или имя класса, если plugin_id не задан.

get_function_prefix() -> str Возвращает префикс пространства имён функций. Возвращает Значение function_prefix или идентификатор плагина, если function_prefix не задан.

initialize(openai=None, bot=None, storage_root: str | None = None) -> None Инициализирует плагин с внешними зависимостями. Аргументы openai — объект клиента OpenAI API bot — объект бота (Telegram или иной) storage_root — путь к корневой директории для хранения данных плагина

close() -> None Освобождает ресурсы при завершении работы плагина.

on_startup(application: Any) -> None Асинхронный хук, вызываемый после готовности Telegram-приложения. Аргументы application — объект приложения Telegram

get_bot_language() -> str Определяет язык бота из конфигурации OpenAI. Возвращает Код языка из конфигурации или "en" по умолчанию.

t(key: str, **kwargs: Any) -> str Возвращает локализованный текст по ключу с подстановкой параметров. Аргументы key — ключ локализованной строки kwargs — именованные параметры для форматирования строки Возвращает Локализованный текст с применённой подстановкой.

get_source_name() -> str Возвращает название источника плагина.

get_spec() -> [Dict] Возвращает спецификации функций в формате JSON Schema согласно документации OpenAI.

execute(function_name: str, helper: Any, **kwargs: Optional[Dict[str, Any]]) -> Dict Выполняет функцию плагина и возвращает JSON-сериализуемый результат. Аргументы function_name — имя выполняемой функции helper — вспомогательный объект для выполнения функции kwargs — опциональные именованные аргументы, могут быть частичными Возвращает JSON-сериализуемый словарь с результатом выполнения.

get_commands() -> List[Dict] Возвращает список поддерживаемых Telegram-команд. Возвращает Список словарей с ключами: command (str), description (str), args (str, опционально), handler (callable), handler_kwargs (dict).

get_message_handlers() -> List[Dict] Возвращает список обработчиков сообщений. Возвращает Список словарей с конфигурацией обработчиков.

get_inline_handlers() -> List[Dict] Возвращает список обработчиков inline-запросов. Возвращает Список словарей с конфигурацией обработчиков.
