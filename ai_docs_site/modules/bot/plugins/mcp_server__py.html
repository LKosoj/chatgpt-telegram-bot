bot/plugins/mcp_server

Источник: bot/plugins/mcp_server.py

Модуль интеграции с MCP (Model Context Protocol) серверами. Поддерживает два транспорта — stdio для локальных процессов и HTTP для удалённых серверов. Предоставляет API для регистрации серверов, получения инструментов, выполнения функций с проверкой прав доступа, а также Telegram-интерфейс для управления серверами. Реализована ролевая модель с разделением прав между обычными пользователями и администраторами.

Ключевые структуры данных servers — словарь зарегистрированных MCP-серверов с их конфигурацией (имя, транспорт, URL/команда, API-ключ, описание, список инструментов) sessions — словарь активных клиентских сессий (имя сервера → ClientSession) connections — словарь активных соединений (имя сервера → MCPConnection) tools — реестр доступных инструментов (имя функции → MCPTool)

class MCPServerPlugin Плагин для поддержки сторонних MCP серверов.

Поля openai — ссылка на OpenAI-клиент bot — ссылка на объект бота servers — конфигурации зарегистрированных серверов config_path — путь к файлу конфигурации JSON admin_ids — список ID администраторов allowed_users — список разрешённых пользователей (пустой = все) sessions — активные клиентские сессии logger — логгер для записи событий плагина
