bot/plugins/haiper_image_to_video

Источник: bot/plugins/haiper_image_to_video.py

Модуль интеграции с API Haiper для генерации видео из статичных изображений в Telegram-боте. Предоставляет плагин HaiperImageToVideoPlugin с асинхронной обработкой очереди задач, диалоговым конструктором промптов, интерактивным выбором параметров анимации и автоматической доставкой результатов пользователям.

Ключевые структуры данных TempFileManager — контекстный менеджер для создания и автоматической очистки временных файлов. TaskStatus — перечисление состояний задачи генерации (PENDING, PROCESSING, COMPLETED, FAILED). VideoTask — dataclass, представляющая задачу генерации видео с метаданными пользователя, статусом и результатом. WAITING_PROMPT — константа состояния ConversationHandler для ожидания дополнительного текста промпта. INLINE_EFFECT_IDS — список идентификаторов эффектов, доступных для inline-запросов. UserSettings — словарь настроек пользователя (style, effect, preset, file_id, base_prompt).

class TempFileManager Контекстный менеджер для автоматической работы с временными файлами. Поля suffix — расширение временного файла (опционально). temp_file — объект временного файла. path — путь к созданному временному файлу. TaskStatus — перечисление состояний задачи генерации видео.

VideoTask — dataclass задачи генерации видео. Поля task_id — уникальный идентификатор задачи. user_id — идентификатор пользователя Telegram. chat_id — идентификатор чата Telegram. file_id — идентификатор файла изображения в Telegram. prompt — текстовый промпт для генерации. status — текущее состояние задачи (TaskStatus). result — данные результата при успешном выполнении (опционально). error — сообщение об ошибке при неудаче (опционально).
