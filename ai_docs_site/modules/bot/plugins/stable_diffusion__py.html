bot/plugins/stable_diffusion

Источник: bot/plugins/stable_diffusion.py

Модуль реализует плагин StableDiffusionPlugin для интеграции с LLMGateway на основе унифицированного интерфейса Plugin. Поддерживает генерацию изображений по текстовому промпту и их редактирование по URL, обеспечивая обратную совместимость с legacy-кодом, ориентированным на Stable Diffusion.

Ключевые структуры данных нет

class StableDiffusionPlugin Плагин генерации и редактирования изображений через LLMGateway с обратной совместимостью. Методы get_source_name() -> str — Возвращает идентификатор источника плагина. get_spec() -> [Dict] — Возвращает спецификацию доступных инструментов плагина в формате JSON Schema. async _generate_image(helper, prompt: str) -> tuple[str, str] — Генерирует изображение по текстовому промпту через LLMGateway. async _edit_image(helper, prompt: str, image_url: str) -> Редактирует изображение по URL через LLMGateway. async execute(function_name: str, helper, **kwargs) -> Dict — Диспетчеризует вызов к соответствующему инструменту плагина.

get_source_name() -> str Возвращает строковый идентификатор источника плагина. Возвращает Строка "LLMGateway Image".

get_spec() -> [Dict] Возвращает спецификацию инструментов плагина для регистрации в системе плагинов. Возвращает Список из двух словарей с описанием инструментов stable_diffusion и edit_image в формате JSON Schema, включающий имена, описания, параметры и списки обязательных полей.

async _generate_image(helper, prompt: str) -> tuple[str, str] Генерирует изображение через LLMGateway на основе текстового описания. Аргументы helper — Вспомогательный объект с клиентом LLMGateway и конфигурацией, содержащий client, config. prompt — Текстовое описание желаемого изображения. Возвращает Кортеж (image_value, image_format) с данными изображения и его форматом, полученный через extract_image_result.

async _edit_image(helper, prompt: str, image_url: str) -> tuple[str, str] Редактирует существующее изображение через LLMGateway по указанному URL. Аргументы helper — Вспомогательный объект с клиентом LLMGateway, содержащий gateway_client. prompt — Инструкции по редактированию изображения. image_url — URL изображения для редактирования (HTTP(S), data URL или ссылка). Возвращает Кортеж (image_value, image_format) с отредактированными данными изображения и его форматом, полученный через extract_image_result.

async execute(function_name: str, helper, **kwargs) -> Dict Диспетчеризует вызов инструмента плагина и формирует результат для ответа пользователю. Аргументы function_name — Имя вызываемого инструмента ("stable_diffusion" или "edit_image"). helper — Вспомогательный объект с клиентами и конфигурацией LLMGateway. kwargs — Именованные аргументы вызова, ожидается prompt и опционально image_url. Возвращает Словарь с ключом "direct_result" (структура с kind, format, value, add_value для фото) или "result" (строка с ошибкой). Исключения Все исключения перехватываются внутри метода, логируются и возвращаются как строка ошибки в поле "result".
