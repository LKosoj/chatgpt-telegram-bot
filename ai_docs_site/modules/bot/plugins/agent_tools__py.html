bot/plugins/agent_tools

Источник: bot/plugins/agent_tools.py

Lightweight agent-support tools for the existing function-calling loop. Provides task planning management, interactive user questioning, artifact delivery, and subagent orchestration with deduplication, TTL, and isolation mechanisms.

Ключевые структуры данных CLOSED_STATUSES — множество терминальных статусов задач: {"completed", "cancelled"} TASK_STATUSES — множество всех допустимых статусов задач: {"pending", "in_progress", "completed", "cancelled"} DELIVERY_ACTION_WORDS — кортеж ключевых слов действий для эвристики доставки артефактов DELIVERY_TARGET_WORDS — кортеж ключевых слов целей для эвристики доставки артефактов MAX_SUBAGENTS — максимальное количество одновременных субагентов (5) MIN_SUBAGENT_TOOL_ROUNDS — минимальное количество раундов инструментов для субагента (10) MAX_SUBAGENT_TOOL_ROUNDS — максимальное количество раундов инструментов для субагента (50) DEFAULT_SUBAGENT_TEMPERATURE — температура по умолчанию для субагента (0.2) SUBAGENT_BLOCKED_FUNCTIONS — множество функций, запрещённых для вызова из субагента
