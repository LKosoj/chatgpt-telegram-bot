bot/plugins/vkusvill

Источник: bot/plugins/vkusvill.py

Adapter plugin for the VkusVill remote MCP server. Provides a set of tools for interacting with VkusVill services including shop search, product search and details, discount listings, analogs, cart link generation, and recipe browsing. Handles MCP protocol communication, argument mapping with defaults, and result transformation for integration into the plugin framework. Supports recipe filtering by categories, complexity, cooking time, preparation method, and allergen exclusion with pagination and sorting capabilities.

Ключевые структуры данных VkusVillPlugin — Main adapter class extending Plugin base class, manages tool registration and remote MCP communication _tools — Class-level dictionary mapping local tool names to remote endpoint configurations including defaults and JSON schema specifications INTERNAL_ARGUMENTS — Set of argument names reserved for internal framework use and excluded from remote calls MCP_PROTOCOL_VERSION — MCP protocol version string for compatibility VKUSVILL_MCP_URL — Base URL for the VkusVill MCP server endpoint
