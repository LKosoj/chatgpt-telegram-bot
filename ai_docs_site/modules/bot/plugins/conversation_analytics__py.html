bot/plugins/conversation_analytics

Источник: bot/plugins/conversation_analytics.py

Плагин для сбора, хранения и анализа статистики чат-диалогов. Модуль предоставляет плагин ConversationAnalyticsPlugin для анализа паттернов диалогов и генерации инсайтов из истории чатов. Собирает статистику по сообщениям, использованию токенов, темам, настроениям и часам активности. Поддерживает временные периоды анализа (день, неделя, месяц) и типы рекомендаций (темы, обучение, формат контента, стиль взаимодействия). Данные хранятся в JSON-файле с автоматической очисткой записей старше 30 дней. Включает инструменты для извлечения технических терминов, анализа предпочтений формата контента и паттернов взаимодействия, а также статистического анализа длины сообщений с выявлением аномалий. Результаты анализа форматируются в читаемый текст с эмодзи-разметкой.

Ключевые структуры данных

conversation_stats — вложенный словарь с ключом chat_id, содержащий списки сообщений, счётчики токенов и тем, оценки настроений и распределение активности по часам суток

TECHNICAL_TERMS — предопределённое множество технических терминов для поиска в сообщениях

patterns — словарь с метриками длины сообщений: ключи 'message_length' (список длин), 'avg_length' (среднее), 'min_length' (минимум), 'max_length' (максимум), 'length_percentages' (процентные доли)

class ConversationAnalyticsPlugin Плагин для анализа паттернов диалогов и генерации инсайтов из истории чатов.

Поля

analytics_dir — Путь к каталогу хранения аналитических данных.

analytics_file — Путь к JSON-файлу со статистикой.

conversation_stats — Вложенный словарь статистики по чатам.

initialize(self, openai=None, bot=None, storage_root: str | None = None) -> None — Настраивает путь хранения данных на основе storage_root и перезагружает статистику.

get_source_name(self) -> str — Возвращает идентификатор источника плагина.

get_spec(self) -> List[Dict] — Возвращает спецификации доступных функций для интеграции с системой плагинов.

load_stats(self) -> Dict — Загружает статистику из файла с инициализацией отсутствующих структур.

save_stats(self) — Сохраняет текущую статистику в JSON-файл.

update_stats(self, chat_id, message_data: Dict) — Обновляет статистику чата новыми данными сообщения и выполняет очистку устаревших записей.

execute(self, function_name: str, helper, **kwargs) -> Dict — Выполняет запрошенную аналитическую функцию асинхронно.

_extract_technical_terms(messages) Извлекает технические термины из списка сообщений.

Аргументы

messages — Список словарей сообщений с ключом 'text'.

Возвращает

Отсортированный список найденных уникальных технических терминов.

_analyze_format_preferences(messages) Анализирует предпочтения пользователя по формату контента.

Аргументы

messages — Список словарей сообщений.

Возвращает

Словарь с количеством сообщений по категориям: 'text', 'code', 'images', 'voice'.

_analyze_interaction_patterns(messages) Анализирует паттерны взаимодействия пользователя в диалоге.

Аргументы

messages — Список словарей сообщений.

Возвращает

Словарь со статистикой: средняя длина сообщения, частота вопросов, использование команд.

analyze_message_patterns(messages) Анализирует паттерны длины сообщений из списка коммуникационных записей.

Аргументы

messages — Список словарей с коммуникационными данными, каждый элемент содержит поле 'length' с длиной сообщения.

Возвращает

Словарь со статистическими метриками длины сообщений.

Исключения

ValueException — Если список messages пуст или отсутствует поле 'length' в записях.

calculate_length_percentages(patterns) Вычисляет процентное распределение длин сообщений относительно среднего значения.

Аргументы

patterns — Словарь с метриками длины, содержащий ключ 'message_length' со списком длин и 'avg_length' со средним значением.

Возвращает

Обновленный словарь с добавленным ключом 'length_percentages', содержащим список процентных отношений каждой длины к среднему.

Исключения

ZeroDivisionError — Если средняя длина сообщений равна нулю.
