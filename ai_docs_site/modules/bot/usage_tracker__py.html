bot/usage_tracker

Источник: bot/usage_tracker.py

Модуль отслеживания использования ресурсов пользователями бота. Предоставляет класс UsageTracker для учёта потребления токенов чата, изображений, vision-токенов, TTS-символов, транскрипции и других ресурсов с сохранением истории в JSON-файлы. Поддерживает расчёт дневных, месячных и общих затрат с применением модель-зависимых тарифов и агрегации статистики за различные периоды.

Ключевые структуры данных UsageTracker — класс для отслеживания использования ресурсов пользователем, хранит данные в JSON-формате с полями user_name, current_cost и usage_history self.usage — словарь с историей использования и текущими затратами, хранящийся в JSON-файле пользователя self.user_file — путь к JSON-файлу, в котором сохраняется статистика использования

year_month(date_str) Извлекает строку год-месяц из даты в формате 'YYYY-MM'. Аргументы date_str — строка или объект даты, из которого извлекается год-месяц Возвращает Строку формата 'YYYY-MM' (первые 7 символов)

class UsageTracker Класс для отслеживания дневного/месячного использования ресурсов пользователем.

Поля user_id — идентификатор пользователя (Telegram ID) logs_dir — директория для хранения файлов использования, по умолчанию "usage_logs" user_file — путь к JSON-файлу использования пользователя usage — словарь с данными использования в формате JSON

UsageTracker.init(self, user_id, user_name, logs_dir="usage_logs") Инициализирует UsageTracker для пользователя с текущей датой. Загружает данные использования из файла или создаёт новые. Аргументы user_id — Telegram ID пользователя user_name — имя пользователя Telegram logs_dir — путь к директории логов использования, по умолчанию "usage_logs"

UsageTracker.add_chat_tokens(self, tokens, tokens_price=0.002) Добавляет использованные токены чата из запроса в историю использования и обновляет текущие затраты. Аргументы tokens — общее количество токенов, использованных в последнем запросе tokens_price — цена за 1000 токенов, по умолчанию 0.002

UsageTracker.get_current_token_usage(self) Возвращает количество токенов чата, использованных за сегодня и за текущий месяц. Возвращает Кортеж (usage_day, usage_month) с количеством токенов за день и месяц

UsageTracker.add_image_request(self, image_size, image_prices="0.016,0.018,0.02") Добавляет запрос изображения в историю использования и обновляет текущие затраты. Аргументы image_size — запрошенный размер изображения ("256x256", "512x512" или "1024x1024") image_prices — цены для изображений разных размеров в формате строки, по умолчанию "0.016,0.018,0.02"

UsageTracker.get_current_image_count(self) Возвращает количество изображений, запрошенных за сегодня и за текущий месяц. Возвращает Кортеж (usage_day, usage_month) с количеством изображений за день и месяц

UsageTracker.add_vision_tokens(self, tokens, vision_token_price=0.01) Добавляет vision-токены в историю использования и обновляет текущие затраты. Аргументы tokens — общее количество vision-токенов, использованных в последнем запросе vision_token_price — цена за 1000 vision-токенов, по умолчанию 0.01

UsageTracker.get_current_vision_tokens(self) Возвращает количество vision-токенов за сегодня и за текущий месяц. Возвращает Кортеж (tokens_day, tokens_month) с количеством vision-токенов за день и месяц

UsageTracker._tts_price_for_model(self, tts_model, tts_prices) Возвращает цену для указанной TTS-модели из списка доступных моделей. Аргументы tts_model — строка с идентификатором модели TTS tts_prices — список или кортеж цен для моделей TTS Возвращает Числовое значение цены за 1000 символов для указанной модели или цену первой модели по умолчанию Исключения IndexError — если индекс модели выходит за границы списка цен

UsageTracker.add_tts_request(self, text_length, tts_model, tts_prices) Регистрирует запрос на синтез речи, обновляет текущие затраты и историю использования по символам. Аргументы text_length — количество символов для синтеза речи tts_model — идентификатор используемой TTS-модели tts_prices — список цен для расчёта стоимости

UsageTracker.get_current_tts_usage(self) Возвращает суммарное количество символов, преобразованных в речь, за сегодня и текущий месяц. Возвращает Кортеж из двух целых чисел: (characters_day, characters_month)

UsageTracker.add_transcription_seconds(self, seconds, minute_price=0.006) Добавляет секунды транскрипции в историю использования и обновляет текущие затраты. Аргументы seconds — общее количество секунд в последнем запросе транскрипции minute_price — цена за минуту транскрипции, по умолчанию 0.006

UsageTracker.get_current_transcription_duration(self) Возвращает длительность транскрибированного аудио за сегодня и текущий месяц в минутах и секундах. Возвращает Кортеж из четырёх значений: (minutes_day, seconds_day, minutes_month, seconds_month)

UsageTracker.add_current_costs(self, request_cost) Добавляет стоимость запроса к совокупным затратам за день, месяц и всё время с учётом даты последнего обновления. Аргументы request_cost — стоимость текущего запроса в USD

UsageTracker.get_current_cost(self) Возвращает совокупные затраты за текущий день, месяц и всё время. Возвращает Словарь с ключами "cost_today", "cost_month", "cost_all_time"

UsageTracker.initialize_all_time_cost(self, tokens_price=0.002, image_prices="0.016,0.018,0.02", minute_price=0.006, vision_token_price=0.01, tts_prices='0.015,0.030') Вычисляет полную стоимость всех запросов в истории использования на основе накопленной статистики и заданных тарифов. Аргументы tokens_price — цена за 1000 токенов, по умолчанию 0.002 image_prices — строка с ценами изображений разных размеров через запятую, по умолчанию "0.016,0.018,0.02" minute_price — цена за минуту транскрипции, по умолчанию 0.006 vision_token_price — цена за 1000 токенов vision, по умолчанию 0.01 tts_prices — строка с ценами TTS-моделей через запятую, по умолчанию '0.015,0.030' Возвращает Общая стоимость всех запросов в истории в USD

UsageTracker.all_time_cost(self) Вычисляет полную совокупную стоимость всех типов потреблённых ресурсов. Возвращает число — суммарная стоимость token_cost + transcription_cost + image_cost + vision_cost + tts_cost, округлённая до 2 знаков Исключения KeyError — при отсутствии обязательных ключей в self.usage['usage_history'] ValueError — при некорректном формате данных в истории использования или ценах
