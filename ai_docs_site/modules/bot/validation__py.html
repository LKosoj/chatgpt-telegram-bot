bot/validation

Источник: bot/validation.py

Модуль предоставляет средства валидации конфигурации OpenAI и аргументов функций на основе JSON Schema. Реализует проверку обязательных полей и типов конфигурации OpenAI, а также валидацию аргументов функций с форматированием понятных сообщений об ошибках.

validate_openai_config(config: Dict[str, Any]) -> None Проверяет, что конфигурация OpenAI содержит все обязательные ключи с корректными типами. Аргументы config — словарь с конфигурацией OpenAI Исключения ValueError — если отсутствуют обязательные ключи или тип значения не соответствует ожидаемому

validate_function_args(spec: Dict[str, Any], args: Dict[str, Any]) -> List[str] Валидирует аргументы функции по JSON Schema из спецификации и возвращает список сообщений об ошибках. Аргументы spec — спецификация функции, содержащая схему параметров args — переданные аргументы функции для проверки Возвращает Список строк с описанием ошибок валидации; пустой список если ошибок нет

_format_validation_error(error) -> str Форматирует объект ошибки jsonschema в человекочитаемое сообщение. Аргументы error — объект ошибки валидации jsonschema Возвращает Отформатированная строка с описанием ошибки

_join_path(path) -> str Преобразует путь ошибки jsonschema в строковое представление с точечной нотацией. Аргументы path — последовательность ключей/индексов пути в структуре данных Возвращает Строка пути, например "foo.bar[0].baz"

_required_property_name(message: str) -> str Извлекает имя обязательного свойства из сообщения об ошибке "required". Аргументы message — исходное сообщение об ошибке валидации required Возвращает Извлечённое имя свойства или исходное сообщение, если извлечь не удалось
