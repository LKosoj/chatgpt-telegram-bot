bot/hindsight_client

Источник: bot/hindsight_client.py

Клиентская библиотека для взаимодействия с Hindsight API — сервисом семантической памяти. Предоставляет асинхронный HTTP-клиент для операций recall (поиск по памяти), retain (сохранение), list (перечисление) и stats (статистика) в рамках именованных банков памяти. Включает вспомогательную функцию форматирования результатов поиска в человекочитаемый вид.

Ключевые структуры данных HindsightError — Базовое исключение для ошибок клиента Hindsight.

class HindsightClient Асинхронный HTTP-клиент для Hindsight API. Поля base_url — Базовый URL сервера Hindsight (без завершающего слэша). api_token — Токен для авторизации через заголовок Authorization. namespace — Пространство имён для изоляции банков памяти (по умолчанию "default"). HindsightClient.init(self, base_url: str, api_token: str = "", *, namespace: str = "default", timeout: float = 30.0, transport: httpx.AsyncBaseTransport | None = None) -> None Инициализация клиента с параметрами подключения. Аргументы base_url — Базовый URL сервера Hindsight. api_token — Токен для авторизации (опционально). namespace — Пространство имён для изоляции банков (по умолчанию "default"). timeout — Таймаут HTTP-запросов в секундах (по умолчанию 30.0). transport — Пользовательский транспорт httpx (опционально).

HindsightClient.enabled(self) -> bool Проверяет, сконфигурирован ли base_url (клиент активен). Возвращает True, если base_url задан и не пуст; иначе False.

HindsightClient.close(self) -> None Закрывает HTTP-соединение и освобождает ресурсы.

HindsightClient._headers(self) -> dict[str, str] Формирует заголовки запроса с Content-Type, Accept и опциональным Authorization. Возвращает Словарь HTTP-заголовков.

HindsightClient._bank_path(self, bank_id: str, suffix: str) -> str Строит путь к ресурсу банка с URL-кодированием идентификатора. Аргументы bank_id — Идентификатор банка памяти. suffix — Относительный путь ресурса внутри банка. Возвращает Абсолютный путь API вида /v1/{namespace}/banks/{bank_id}/{suffix}.

HindsightClient.request(self, method: str, path: str, *, json_payload: dict[str, Any] | None = None, params: dict[str, Any] | None = None, timeout: float | None = None) -> dict[str, Any] Выполняет HTTP-запрос к API с обработкой ошибок и валидацией ответа. Аргументы method — HTTP-метод (GET, POST и т.д.). path — Относительный путь ресурса. json_payload — Тело запроса в формате JSON (опционально). params — Query-параметры запроса (опционально). timeout — Переопределение таймаута для конкретного запроса (опционально). Возвращает Распарсенный JSON-ответ в виде словаря. Исключения HindsightError — Если base_url не сконфигурирован, запрос завершился с кодом ≥400, или ответ не является корректным JSON-объектом.

HindsightClient.recall(self, bank_id: str, query: str, *, budget: str = "mid", max_tokens: int = 4096, memory_types: list[str] | None = None, trace: bool = False) -> dict[str, Any] Выполняет семантический поиск по памяти банка. Аргументы bank_id — Идентификатор банка памяти. query — Поисковый запрос (семантический). budget — Бюджет вычислений (по умолчанию "mid"). max_tokens — Максимальное количество токенов в ответе (по умолчанию 4096). memory_types — Фильтр по типам памяти (опционально). trace — Включить трассировку запроса (по умолчанию False). Возвращает JSON-ответ API с результатами семантического поиска.

HindsightClient.retain_memories(self, bank_id: str, items: list[dict[str, Any]], *, async_store: bool = True) -> dict[str, Any] Сохраняет новые фрагменты памяти в банк. Аргументы bank_id — Идентификатор банка памяти. items — Список фрагментов памяти для сохранения. async_store — Асинхронное сохранение без ожидания индексации (по умолчанию True). Возвращает JSON-ответ API с результатами операции.

HindsightClient.list_memories(self, bank_id: str, *, limit: int = 20, offset: int = 0, query: str | None = None, memory_type: str | None = None) -> dict[str, Any] Перечисляет сохранённые фрагменты памяти с фильтрацией. Аргументы bank_id — Идентификатор банка памяти. limit — Максимальное количество результатов (по умолчанию 20). offset — Смещение для пагинации (по умолчанию 0). query — Текстовый поисковый фильтр (опционально). memory_type — Фильтр по типу памяти (опционально). Возвращает JSON-ответ API со списком фрагментов памяти.

HindsightClient.stats(self, bank_id: str) -> dict[str, Any] Получает статистику банка памяти. Аргументы bank_id — Идентификатор банка памяти. Возвращает JSON-ответ API со статистикой банка.

format_recall_results(data: dict[str, Any], *, max_items: int = 8) -> str Форматирует результаты семантического поиска в человекочитаемый markdown-список. Аргументы data — JSON-ответ от recall с полем results. max_items — Максимальное количество элементов для вывода (по умолчанию 8). Возвращает Строка с markdown-списком вида "- text (type=...; when=...; context=...)" или пустая строка.
