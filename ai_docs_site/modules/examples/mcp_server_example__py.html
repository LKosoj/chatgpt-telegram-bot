examples/mcp_server_example

Источник: examples/mcp_server_example.py

Модуль реализует пример MCP-сервера на базе FastAPI для интеграции с Telegram-ботом. Предоставляет REST API для получения списка доступных инструментов (погода, конвертация валют, генерация идентификаторов) и их выполнения по запросу. Включает модели данных Pydantic, in-memory хранилища примерных данных и базовую авторизацию через заголовки.

Ключевые структуры данных FunctionArguments — Модель аргументов для вызова функции с разрешёнными дополнительными полями (extra="allow"). FunctionCall — Модель запроса на вызов функции, содержащая имя функции и аргументы. ToolSpec — Модель спецификации инструмента с именем, описанием и параметрами в формате JSON Schema.

async def get_weather(city: str) -> Dict Получить текущую погоду для указанного города из in-memory хранилища. Аргументы city — Название города для получения погоды. Возвращает Словарь с данными о погоде (температура, условия, влажность, timestamp) или ошибку со списком доступных городов.

async def convert_currency(amount: float, from_currency: str, to_currency: str) -> Dict Конвертировать сумму из одной валюты в другую по фиксированным курсам. Аргументы amount — Сумма для конвертации. from_currency — Исходная валюта (USD, EUR, GBP, RUB). to_currency — Целевая валюта (USD, EUR, GBP, RUB). Возвращает Словарь с исходной суммой, сконвертированной суммой, курсом обмена и timestamp.

async def generate_id(prefix: str = "") -> Dict Сгенерировать уникальный идентификатор на базе UUID с опциональным префиксом. Аргументы prefix — Префикс для идентификатора (по умолчанию пустая строка). Возвращает Словарь с сгенерированным идентификатором и timestamp.

@app.get("/") async def root() Корневой маршрут, возвращающий базовую информацию о сервере. Возвращает Словарь с именем, описанием, версией сервера и количеством доступных инструментов.

@app.get("/tools") async def get_tools(authorization: Optional[str] = Header(None)) Получить список доступных инструментов в формате JSON Schema. Аргументы authorization — Опциональный токен авторизации из заголовка. Возвращает Список словарей с описанием доступных инструментов.

@app.post("/execute") async def execute_function(call: FunctionCall, authorization: Optional[str] = Header(None)) Выполнить указанную функцию с переданными аргументами после проверки авторизации. Аргументы call — Модель FunctionCall с именем функции и аргументами. authorization — Опциональный токен авторизации из заголовка. Возвращает Результат выполнения вызванной функции. Исключения HTTPException(401) — Если требуется авторизация (REQUIRE_AUTH=true) и токен отсутствует. HTTPException(400) — Если отсутствует обязательный параметр для вызываемой функции.

HTTPException(status_code=404, detail=f"Функция '{call.name}' не найдена") Возбуждает исключение при запросе несуществующей функции. Аргументы status_code — HTTP-код состояния 404 Not Found. detail — Описание ошибки с указанием запрошенного имени функции.

uvicorn.run(app, host="0.0.0.0", port=port) Запускает ASGI-сервер Uvicorn для обработки входящих запросов. Аргументы app — Экземпляр ASGI-приложения. host — Адрес привязки сервера, по умолчанию "0.0.0.0". port — Номер порта, по умолчанию 8885 (переопределяется переменной окружения PORT).
